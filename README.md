# 📋 CPF ธารเกษม — Dashboard ค่าใช้จ่าย & สวัสดิการ

แดชบอร์ดสรุปผล **ค่าใช้จ่าย (OE)** และ **สวัสดิการพนักงาน** ของโรงงานผลิตอาหารสัตว์ธารเกษม (CPF)
เป็น **standalone HTML** เปิดในเบราว์เซอร์ได้ทันที ดึงข้อมูลสดจาก **Google Sheets** และ **Export PowerPoint** ได้

> 🎨 ธีมเขียว CPF · ฟอนต์ Sarabun · ภาษาไทย · ปีงบประมาณ พ.ศ. 2569

---

## 🚀 เริ่มใช้งานเร็ว

- เปิดไฟล์ `index.html` ด้วยเบราว์เซอร์ → เลือกแดชบอร์ดที่ต้องการ
- ครั้งแรกจะแสดง **ข้อมูลตัวอย่าง (Demo)** ทันที — ไม่ต้องตั้งค่าอะไร
- เมื่อพร้อมใช้ข้อมูลจริง กดปุ่ม **🔗 เชื่อม Google Sheets** แล้ววางลิงก์ชีต

หรือเปิดผ่าน URL ของ GitHub Pages (ดูหัวข้อ [Deploy](#-deploy-ขึ้น-github-pages))

## 🔗 เชื่อมต่อ Google Sheets (ข้อมูลสด)

1. เปิด Google Sheets ของคุณ → กด **แชร์ (Share)** → ตั้งเป็น **"ทุกคนที่มีลิงก์ (ผู้อ่าน)"**
2. ตรวจสอบว่ามีแท็บ (sheet) ชื่อ **`RAW_DATA`** ที่มีคอลัมน์ครบตาม [โครงสร้างข้อมูล](#-โครงสร้างข้อมูล-raw_data)
3. ในแดชบอร์ด กดปุ่ม **🔗 เชื่อม Google Sheets** → วางลิงก์ → กด **บันทึก & ดึงข้อมูล**
4. กด **🔄 รีเฟรช** เมื่อต้องการดึงข้อมูลล่าสุด

> ระบบดึงข้อมูลผ่าน **published CSV** ของ Google (`gviz/tq?tqx=out:csv`) ไม่ต้องใช้ API key
> การตั้งค่าจะถูกจำไว้ในเบราว์เซอร์ (localStorage) — ครั้งต่อไปเปิดมาก็ดึงให้อัตโนมัติ

**ถ้าต่อชีตไม่ได้** ระบบจะแจ้งเตือนและกลับไปแสดงข้อมูลตัวอย่างให้อัตโนมัติ
หรือใช้ทางเลือก **📁 อัปโหลดไฟล์ Excel (.xlsx)** ในหน้าต่างเดียวกันก็ได้ (อ่านแท็บ `RAW_DATA`)

|------|---------|---------|
| 1 | เดือน | `2569-01` |

ดูตัวอย่างจริงได้ในโฟลเดอร์ [`sample-data/`](./sample-data) — คัดลอกไปวางใน Google Sheets ได้เลย

#### 🕒 OT (ชั่วโมงล่วงเวลา)

นอกจากแท็บ `RAW_DATA` แบบยาว แดชบอร์ดสวัสดิการยังรองรับ **ฟอร์มตารางแบบกว้าง (wide)** — 1 พนักงาน/แถว
มีคอลัมน์ OT 7 อัตรา (วันทำงานปกติ 1/1.25/1.5 เท่า · วันหยุด 1/2/2.5/3 เท่า) โดยกรอกเป็น **จำนวนชั่วโมง**

- ระบบรวมชั่วโมง OT ทุกอัตราเป็น **"OT รวม (ชั่วโมง)"** แสดง **แยกออกจากยอดเงิน** (KPI/กราฟ/ตาราง Top คน OT เป็นหน่วยชั่วโมง)
- ต้องมีเซลล์ระบุ **เดือน** (รูปแบบ `2569-01`) อยู่ในชีต
- ดาวน์โหลดเทมเพลตพร้อมใช้ได้จากแชต (ไฟล์ `CPF_ธารเกษม_สวัสดิการ_OT_ฟอร์ม.xlsx`)

**กติกาข้อมูล**
- ปีเป็น **พ.ศ.** รูปแบบเดือน `2569-01` ถึง `2569-12`
- ตัวเลขใส่ comma ได้ (`1,234`) ระบบจะตัดให้เอง
- แถวหัวตาราง / แถวยอด 0 จะถูกข้ามอัตโนมัติ

## 📊 Export PowerPoint

กดปุ่ม **📊 Export PowerPoint** ในแต่ละแดชบอร์ด จะได้ไฟล์ `.pptx` 5 สไลด์ตามตัวกรองที่เลือกอยู่
(หน้าปก → KPI → กราฟ/ตารางหลัก) ใช้ฟอนต์ **Sarabun** — หากเครื่องที่เปิด PowerPoint ไม่มีฟอนต์นี้
ระบบจะแทนด้วยฟอนต์ไทยอื่นให้อัตโนมัติ ([ติดตั้ง Sarabun ฟรี](https://fonts.google.com/specimen/Sarabun))

## 🌐 Deploy ขึ้น GitHub Pages

**วิธีที่ 1 — GitHub Actions (อัตโนมัติ, แนะนำ)**
1. ไปที่ **Settings → Pages → Build and deployment → Source** เลือก **"GitHub Actions"**
2. ทุกครั้งที่ push โค้ด workflow `.github/workflows/deploy-pages.yml` จะ deploy ให้อัตโนมัติ
3. URL จะอยู่ที่ `https://<username>.github.io/-AI/`

**วิธีที่ 2 — Deploy from branch (ง่ายสุด ไม่ใช้ Actions)**
1. ไปที่ **Settings → Pages → Source** เลือก **"Deploy from a branch"**
2. เลือก branch ที่ต้องการ + โฟลเดอร์ **`/ (root)`** แล้ว Save

> ใช้โครงสร้าง URL เป็นภาษาอังกฤษ (`/oe/`, `/welfare/`) เพื่อเลี่ยงปัญหาชื่อไฟล์ภาษาไทย
